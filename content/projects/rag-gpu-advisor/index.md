-Implementing intelligent URL and RFP classification using LLM agents (Ollama + LangChain) combined with ML similarity methods like TF-IDF, Jaccard similarity, and vector embeddings.
-Developing scalable backend using ChromaDB, Redis, FastAPI, and Docker, delivering prioritized RFPs autonomosly.
