### Overview
Developed a 6-DOF robotic manipulator with AI-based perception and point-cloud-driven mapping.

### Key Work
- YOLOv8 instance segmentation for payload surface extraction  
- Point cloud processing using PCL and Open3D  
- OctoMap generation and TF calculation  
- ROS + MoveIt integration for motion planning  

### Tech Stack
YOLOv8 · OpenCV · Open3D · PCL · ROS · MoveIt · Python · C++
